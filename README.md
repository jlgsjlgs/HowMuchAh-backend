# Splitwise Clone - Backend

A backend API for a Splitwise-like expense splitting application built with Spring Boot.

## Tech Stack

- **Java 21** (Amazon Corretto)
- **Spring Boot 3.x**
- **Spring Security** with OAuth2
- **Spring Data JPA**
- **PostgreSQL** (Supabase)
- **JWT** for authentication

## Features (Planned)

- User authentication via Google OAuth2
- Create and manage groups
- Add and split expenses
- Track balances and settlements
- Real-time expense calculations

## Prerequisites

- Java 21 or higher
- Maven 3.x
- PostgreSQL database
